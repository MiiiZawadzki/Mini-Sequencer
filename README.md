# Mini-Sequencer
Simple audio sequencer written in python that lets you create multiple projects with up to 15 different sounds.

## Technologies
Project is created with:
* PyGames
* Python 3
* Asyncio

## Key features
* Allows to arrange a sequence of 15 sounds
* Changeable length of sequence
* Two color themes
* Loading and saving projects
* Loop playback capability
* Easy jump to any point in the sequence by dragging the playback cursor.

## Usage
* Installing dependencies:
```sh
pip install -r requirements.txt
```
* Running app:
```sh
python main.py
```
